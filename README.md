# My dotfiles use on your own resposibility

These dotfiles inspired by [dotfiles](https://dotfiles.github.io/)

These dotfiles project aim to provide a faster way to setup PopOS

- VSCode and Atom
- Programming languages and tools
- Shell configuration and plugins
- Git configuration

## Instalation

On fresh PopOS instalation:

```sh
 git clone https://github.com/Topplestack/dotfiles.git
 sudo chmod +x dotfiles
 ./dotfiles
```
